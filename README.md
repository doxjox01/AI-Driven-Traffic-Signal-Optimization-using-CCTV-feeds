# AI Traffic Signal Optimization 🚦

This project is an AI-based smart traffic signal system using YOLOv8 and OpenCV.

## Features
- Multi-lane traffic analysis
- Fair signal distribution
- Occupancy-based congestion detection
- Real-time simulation

## Tech Used
- Python
- OpenCV
- YOLOv8
